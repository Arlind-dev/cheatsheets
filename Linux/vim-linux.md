### Vi Mode

| **vim - lin**        |                                 |
|----------------------|---------------------------------|
| **Kombination**      | **Anwendung /   Beschreibung**  |
| esc                  | exit insert mode                |
| i                    | start insert mod                |
| u                    | undo the last action            |
| :q                   | quit                            |
| :q!                  | quit without saving             |
| :sav                 | save                            |
| :wq                  | save and quit                   |
